# Privacy Policy

## Overview

The Cashfree Payments - Dify Plugin collects and processes user data to enable secure payment and payout operations via Cashfree APIs within Dify.ai workflows.

## Information We Collect

- **Authentication Credentials**
    - Cashfree Client ID
    - Cashfree Client Secret
    - Bearer Token
    - API Version
    - Environment (sandbox/production)

- **Transaction Data**
    - Order details (amount, currency, customer info, order ID, notes)
    - Refund details (order ID, refund amount, refund ID, notes)
    - Payment link details (link ID, amount, currency, purpose, customer info)
    - Cashgram details (cashgram ID, amount, recipient info, expiry, remarks)

## How We Use Your Information

- Authenticate and authorize API requests to Cashfree services
- Create, manage, and track payment orders, refunds, payment links, and Cashgrams
- Facilitate payment and payout operations within Dify.ai workflows
- Notify users and customers about payment and payout status (via email/SMS if enabled)
- Ensure secure and compliant transaction processing

## Data Storage and Security

- Credentials and transaction data are used only for API communication and are not stored beyond session requirements.
- Sensitive information is handled securely and transmitted over encrypted channels.
- No personal or financial data is shared with third parties except as required for payment processing via Cashfree.

## User Control

- Users can choose authentication methods and control which data is provided for each transaction.
- Users may deactivate Cashgrams and cancel payment links at any time.

## Contact

For privacy-related questions, please contact the plugin author at [support@cashfree.com](mailto:support@cashfree.com).
